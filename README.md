# 🚀 Akshit Tyagi - Professional Java Developer Portfolio

Welcome to my portfolio website built using **Java (Spring Boot)** and styled with a premium **Google Gemini Dark Theme**. This project showcases my skills, coding profile stats, and recent backend/full-stack projects.

## ✨ Features
- **Backend Logic:** Built entirely using Java and Spring Boot without requiring heavy database setups (utilises optimized server-side model-view data structures).
- **Gemini AI Dark Theme:** Premium, responsive UI with deep charcoal backgrounds , smooth borders, and modern ambient gradients.
- **Dynamic Profile Integration:** Automatically renders coding stats from LeetCode and CodeChef, along with seamless social connects.

## 🚀 How to Run Locally

### Prerequisites
- JDK 17 or higher installed.
- Apache Maven installed.

### Steps
1. **Clone the repository:**
   ```bash
   git clone https://github.com
   cd https://github.com/tyagiakshit-boop/Portfolio-Project.git
   ```

2. **Build the project:**
   ```bash
   mvn clean compile
   ```

3. **Run the Application:**
    - Open `PortfolioApplication.java` in IntelliJ IDEA and click the **Green Play Button**.
    - **OR** run via terminal:
      ```bash
      mvn spring-boot:run
      ```

4. **Access the Portfolio:**
   Open your browser and navigate to `http://localhost:8080`
